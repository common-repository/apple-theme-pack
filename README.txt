Registers custom post types and custom fields for the Apple theme

== Description ==

**The Apple Theme Pack plugin is meant to be used only with the Apple WordPress theme.**

This plugin registers custom post types and custom fields that are needed in the [Apple theme](http://wordpress.org/themes/sydney/)

== Installation ==

1. Go to Plugins > Add New and search for Apple Theme Pack. Install and activate it.
2. That's it. No configuration needed.
